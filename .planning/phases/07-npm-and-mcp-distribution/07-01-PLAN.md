1. Create `packages/sdk/tsconfig.sdk.json` — shared base TypeScript config for all 14 SDK packages:
   ```json
   {
     "compilerOptions": {
       "target": "ES2022",
       "module": "NodeNext",
       "moduleResolution": "NodeNext",
       "lib": ["ES2022"],
       "outDir": "./dist",
       "rootDir": "./src",
       "strict": true,
       "esModuleInterop": true,
       "skipLibCheck": true,
       "forceConsistentCasingInFileNames": true,
       "declaration": true,
       "declarationMap": true,
       "sourceMap": true
     },

     "exclude": ["node_modules", "dist"]
   }
   ```

2. For each of the 14 SDK packages (bandit, anomaly, bayesian, calibrate, cmaes, decide, ensemble, evolve, forecast, graph, pathfind, risk, simulate, solver), create a per-package `tsconfig.json`:
   ```json
   {
     "extends": "../tsconfig.sdk.json",
     "compilerOptions": {
       "outDir": "./dist",
       "rootDir": "./src"
     }
   }
   ```

3. Update each SDK `package.json` with these fields (preserving all existing fields like name, description, keywords, license, repository, homepage, dependencies):
   - `"main": "dist/index.js"` (was `"src/index.ts"`)
   - `"types": "dist/index.d.ts"` (was `"src/index.ts"`)
   - Add `"exports": { ".": { "import": "./dist/index.js", "types": "./dist/index.d.ts" } }`
   - Add `"files": ["dist"]`
   - Add `"scripts": { "build": "tsc", "prepublishOnly": "npm run build" }`
   - Add `"publishConfig": { "access": "public", "provenance": true }`
   - For the 5 already-published packages (bandit, calibrate, decide, graph, solver): set `"version": "1.1.0"` (cannot republish 1.0.0; the format changed from raw TS to compiled JS)
   - For the 9 unpublished packages: keep `"version": "1.0.0"`

4. Build all 14 SDK packages by running `npx tsc` from within each package directory. Use a for-loop:
   ```bash
   for pkg in packages/sdk/*/; do
     cd "$pkg" && npx tsc && cd -
   done
   ```

5. Verify each package's dist/ contains index.js, index.d.ts, and index.d.ts.map.
