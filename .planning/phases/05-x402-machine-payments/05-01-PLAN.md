       - Import sendProblem and ProblemTypes from '../utils/problem-details'
       - Import FastifyRequest, FastifyReply from 'fastify'
       - The hook factory takes an x402 resource server object (typed loosely to avoid tight coupling to @x402/core internals that may differ between import paths), walletAddress (string), pricePerCall (string like '$0.001'), and network (string like 'eip155:8453')
       - Check `request.headers['payment-signature'] || request.headers['x-payment']` for the payment header
       - If no payment header, return (pass through to next hook)
